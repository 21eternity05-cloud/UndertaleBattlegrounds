# Undertale Battlegrounds

A Roblox Undertale-inspired battlegrounds game built with Rojo.

Current focus:
- Chara and Sans as the first playable characters
- Hollow Snowdin arena
- Combat system with M1s, blocking, guardbreaks, counters, projectiles, ult meter, and cinematic moves

## Current Character Identities

### Chara
Chara is a violent rushdown / execution-pressure character.

She wants to get close, force panic, and punish defensive habits.

Current kit:
- Knife Dash / Knife Lunge
- Red Slash
- Slash Barrage
- Killing Intent
- Ultimate: Erased

### Sans
Sans is a zoner-combo hybrid / slippery control character.

He controls space with bones and aimed threats, then converts mistakes into weird combo routes.

Planned/current kit:
- Bone Shot
- Bone Zone
- Shortcut
- Gaster Blaster
- Ultimate: Bad Time
